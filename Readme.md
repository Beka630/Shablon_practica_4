# Factory Method – Document Management System (Python)

## Описание проекта

Данный проект реализует шаблон проектирования **Factory Method** на языке Python.

Программа предназначена для создания различных типов документов в зависимости от выбора пользователя.

Реализованные типы документов:
- Report
- Resume
- Letter
- Invoice

Создание объектов происходит через фабричный метод без прямого создания конкретных классов в основном коде.

---

## Используемый шаблон проектирования

**Factory Method** — порождающий шаблон, который определяет интерфейс создания объекта, но позволяет подклассам изменять тип создаваемых объектов.

В проекте:

- `Document` — абстрактный продукт
- `Report`, `Resume`, `Letter`, `Invoice` — конкретные продукты
- `DocumentCreator` — абстрактный создатель
- `ReportCreator`, `ResumeCreator`, `LetterCreator`, `InvoiceCreator` — конкретные создатели
